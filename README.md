# 📚 Coaching Institute Website | EduPulse

A comprehensive, trust-building landing page designed for coaching centers, tutors, and educational institutions to showcase courses and testimonials.

---

## 🚀 Overview
**EduPulse** is designed to provide clarity and build authority. This project prioritizes information hierarchy, ensuring that students and parents can quickly find course details, faculty information, and success stories.

---

## ✨ Features
- 📖 **Course Catalog:** Structured grid for multiple academic/skill-based courses.
- 🗣️ **Testimonial Sliders:** Showcasing student success stories to build trust.
- ❓ **FAQ Section:** Dynamic accordion for quick answers.
- 📅 **Schedule Display:** Clear tables for upcoming batches.
- 📞 **Enquiry System:** Lead capture forms for enrollment queries.

---

## 🛠️ Tech Stack
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Runtime:** Vanilla JavaScript (ES6+)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Components:** Modular HTML/JS architecture.

## 🚀 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rajatsec/coaching-institute-site.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start development server:**
   ```bash
   npm run dev
   ```
